# Purchase Pattern Analysis

This project analyzes customer purchase patterns using SQL and Python.

## Tools Used
- SQL (MySQL)
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## Project Structure

- SQL  
  - Week 1: Purchase pattern analysis using SQL queries

- Week 2: Data Cleaning  
  - Cleaned missing values and incorrect data

- Week 3: Exploratory Data Analysis (EDA)  
  - Analyzed sales trends and item-level patterns

- Week 4: Visualization  
  - Created charts and graphs to show insights

## Outcome
This project helps understand customer buying behavior and sales trends.
